# Yet another repo with nestjs

## Description

Personal repo for learning NestJS.

Based on [medium-graphql-nestjs](https://github.com/ramzitannous/medium-graphql-nestjs)

## Installation

```bash
$ npm install
$ cp ./.env.example ./.env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
